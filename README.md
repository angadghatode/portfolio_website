## System Overview

This interface is designed to emulate a vintage gaming console while maintaining modern web standards.

* **Parallax Architecture**: 5-layer horizontal and vertical parallax depth effect.
* **Dynamic UI**: Powered by **NES.css** for that authentic 16-bit look and feel.
* **Performance**: Built on **Vite** with optimized WebP assets to ensure smooth 60FPS animations.
* **Responsive Engine**: Designed to scale across different viewport resolutions.

    ```bash
    npm run dev
    ```

4.  **Deploy to GitHub Pages**
    ```bash
    npm run deploy
    ```
